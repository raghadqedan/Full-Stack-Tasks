# Pro Quiz Website 🎯

A quiz website built with **HTML, CSS, and JavaScript**.  
This project allows users to test their knowledge through  interactive quizzes with a modern UI.

---

## 🚀 Features
- Multiple choice questions
- Progress tracking
- Score calculation
- Responsive design
